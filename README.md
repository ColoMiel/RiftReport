# 🎮 RiftReport — AI-Powered LoL Stats Coach

> Automated League of Legends coaching reports delivered directly to Discord and Email, powered by the Riot API and Mistral AI.

## 📋 Overview

RiftReport is a fully automated n8n workflow that analyzes a player's last 20 ranked games on League of Legends and generates a personalized AI coaching report. The report is sent to Discord as a rich embed and by email.

**Two ways to trigger a report:**
- 💬 Type `/rr RiotID#TAG` in your Discord channel
- 🔗 Send a POST request via webhook

---

## ✨ Features

- 📊 **20-game analysis** — winrate, KDA, CS/min, vision score, kill participation
- 🏆 **Rank tracking** — Solo/Duo and Flex
- 🤖 **AI Coaching** — personalized tips generated by Mistral AI
- 🖼️ **Dynamic champion image** — thumbnail of your most played champion
- 📱 **Discord embed** — rich formatted report sent to your channel
- 📧 **Email report** — full HTML report in your inbox
- ⏱️ **Anti-spam protection** — 5-minute cooldown between reports

## 🔑 Required Secrets

You will need **4 credentials** to run this workflow.

### 1. Riot Games API Key
- Go to [developer.riotgames.com](https://developer.riotgames.com)
- Log in and copy your **Development API Key**
- ⚠️ Dev keys expire every 24 hours — apply for a Production Key for permanent access
- Used in nodes: `GET PUUID`, `GET Rank`, `GET Match IDs`, `GET Match Detail`

```
Header: X-Riot-Token
Value:  RGAPI-xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
```

### 2. Mistral AI API Key
- Go to [console.mistral.ai](https://console.mistral.ai)
- Create an account → **API Keys** → **Create new key**
- Add it as a credential in n8n: **Settings → Credentials → New → Mistral Cloud**
- Used in node: `Mistral Cloud Chat Model`

### 3. Discord Bot Token
- Go to [discord.com/developers/applications](https://discord.com/developers/applications)
- **New Application** → **Bot** → **Reset Token** → copy
- Enable **Message Content Intent** under Privileged Gateway Intents
- Invite the bot to your server via **OAuth2 → URL Generator** (scopes: `bot`, permissions: `Send Messages`, `Read Message History`, `Embed Links`)
- Add it as a credential in n8n: **Settings → Credentials → New → Discord Bot API**
- Used in nodes: `Get many messages`, `Send Discord Embed`

### 4. Email (SMTP)
- Configure your email provider SMTP credentials in n8n
- Add it as a credential: **Settings → Credentials → New → SMTP**
- Used in node: `Send Email Report`

## 🚀 Installation

### Prerequisites
- n8n self-hosted (v2.0+)
- Node.js 18+

### Steps

**1. Import the workflow**
```
n8n → Workflows → Import from file → RiftReport_workflow.json
```

**2. Configure credentials**

Set up the 4 credentials listed above in **n8n Settings → Credentials**.

**3. Update the workflow constants**

In the `Parse Input` node, update the default values:
```javascript
const apiKey = 'YOUR_RIOT_API_KEY';
```

In the `Code Filtre & Parse` node, update the Bot ID:
```javascript
const BOT_ID = 'YOUR_DISCORD_BOT_ID';
```

**4. Configure Discord channel**

In `Get many messages` and `Send Discord Embed` nodes, select your server and channel from the dropdown.

**5. Activate the workflow**

Toggle the workflow to **Active** in the top right corner.

## 🔧 Workflow Nodes

| Node | Type | Description |
|------|------|-------------|
| `Schedule Trigger` | Trigger | Polls Discord every 59 seconds |
| `Webhook - /rr` | Trigger | Manual HTTP trigger |
| `Get many messages` | Discord | Reads last message from channel |
| `Code Filtre & Parse` | Code | Anti-spam + command parsing |
| `Parse Input` | Code | Extracts gameName and tagLine |
| `GET PUUID` | HTTP | Fetches player PUUID from Riot API |
| `GET Rank` | HTTP | Fetches Solo/Duo and Flex rank |
| `GET Match IDs` | HTTP | Fetches last 20 match IDs |
| `Split Match IDs` | Code | Splits IDs for parallel processing |
| `GET Match Detail` | HTTP | Fetches full match data (batched) |
| `Extract Player Stats` | Code | Extracts per-game stats |
| `Aggregate All Games` | Aggregate | Groups all games into one item |
| `Compute Global Stats` | Code | Calculates averages, streaks, trends |
| `Basic LLM Chain` | AI | Sends stats to Mistral for coaching |
| `Extract Coaching Text` | Code | Parses AI response |
| `Send Discord Embed` | Discord | Sends rich embed to channel |
| `Send Email Report` | Email | Sends HTML report by email |

## ⚙️ Configuration

| Parameter | Default | Description |
|-----------|---------|-------------|
| Match count | 20 | Number of games to analyze |
| Schedule interval | 59s | Discord polling frequency |
| Cooldown | 5 min | Minimum time between reports |
| Max message age | 2 min | Ignore commands older than this |
| Batch interval | 1200ms | Delay between Riot API calls |
